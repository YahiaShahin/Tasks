## Task 1 (task1.c)

### Sequence & Loop Functions
- `one_to_n(int num)` - Print numbers 1 to n
- `n_to_one(int num)` - Print numbers n to 1
- `alpha()` - Print alphabet a-z
- `even_to_100()` - Print even numbers 1-100
- `odd_to_100()` - Print odd numbers 1-100

### Summation Functions
- `sum_to_n(int num)` - Sum of numbers 1 to n
- `sum_even_to_n(int num)` - Sum of even numbers up to n
- `sum_odd_to_n(int num)` - Sum of odd numbers up to n

### Number Manipulation Functions
- `multiplication_table(int num_start, int num_end)` - Print multiplication tables
- `num_of_digits(int num)` - Count digits in a number
- `swap_first_and_last_digits(int num)` - Swap first and last digits
- `sum_of_digits(int num)` - Sum all digits
- `product_of_digits(int num)` - Multiply all digits

## Task 2 (task2.c)

### Sequence & Loop Functions
- `one_to_n(int num)` - Print numbers 1 to n
- `n_to_one(int num)` - Print numbers n to 1
- `alpha()` - Print alphabet a-z
- `even_to_100()` - Print even numbers 1-100

### Mathematical Functions
- `power(int base, int exp)` - Calculate base^exponent
- `factors(int num)` - Find all factors of a number
- `factorial(int num)` - Calculate factorial
- `is_prime(int num, int print_result)` - Check if number is prime
- `primes_to_n(int num)` - Print all primes up to n
- `sum_primes_to_n(int num)` - Sum all primes up to n

### Shape Drawing Functions
- `draw_square(int size)` - Draw a square
- `draw_rhombus_right(int size)` - Right-shifted rhombus
- `draw_rhombus_left(int size)` - Left-shifted rhombus
- `draw_right_triangle_left(int size)` - Left-aligned triangle
- `draw_right_triangle_right(int size)` - Right-aligned triangle
- `draw_right_triangle_down_left(int size)` - Inverted left triangle
- `draw_right_triangle_down_right(int size)` - Inverted right triangle
- `draw_pyramid(int size)` - Pyramid pattern
- `draw_inverted_pyramid(int size)` - Inverted pyramid pattern
- `draw_heart(char *prompt, int size)` - Heart shape with centered text
